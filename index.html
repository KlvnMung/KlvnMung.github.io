  1. step 1:Create a new branch in github manually
  2. step 2:choose a file to update
  3. step 3:make changes to the file
  4. step 4:preview changes to check if all is as planned
  5. step 5:commit changes with a commit message
  6. step 6:go to your editor's terminal/cmd and pull changes
  7. check for branches with git branch -r(-r for remote repo only,-a for all repos both local and origin)
